# Fusing-Noisy-MRI-Images
Statistical Digital Signal Processing, TU Delft, Q1
Group Project

# Repository Contents

- `main.m`: main script for noise detection and filtering pipeline
- `report.pdf`: project report
- `MRI_datasets/`: sample MRI images (eye scans)

# Summary

Developed a pipeline to detect and restore outliers in eye MRI scans caused by patient motion and noise, using optimal filtering and spectral estimation techniques. Implemented and compared multiple filters, evaluating image quality improvements quantitatively using SNR.

# Requirements

- MATLAB R2024b
